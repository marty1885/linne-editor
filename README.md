## Build
### The IDE Way
1. clone the proect
2. Get Qt Creator
3. Load the `linne-editor.pro`
4. Configure it
5. Build
6. Copy the `data` folder to your build folder
7. Done!

### Command Line Way
1. clone the proect
2. `cd linne-editor`
3. `qmake .`
4. `make -j4`
5. Done!
